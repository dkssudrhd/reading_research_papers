번역자 메모

번역자 메모: 이 논문은 조류충돌방지 시스템과 직접 관련성이 높지만, 코드와 데이터셋 공개 링크가 확인되지 않아 재현성 검토가 필요합니다. 공개 저장소에는 전문 번역 대신 원문 링크와 라이선스 판단을 남깁니다.

실무 읽기 메모

  • bird detection: 실제 카메라별 새 bbox 크기와 조건별 recall을 먼저 측정합니다.
  • small/tiny object detection: 작은 새가 입력 축소 과정에서 사라지는지 확인합니다.
  • PTZ camera tracking: detector 안정화 후 PTZ 제어를 붙여야 카메라 흔들림이 줄어듭니다.
  • real-time RTSP/video analysis: 저조도 보정은 latency가 크므로 조건부 적용을 검토합니다.
  • YOLO training/data design: 야간, 비, 터빈 배경, 바다 배경을 나눈 validation set이 필요합니다.
  • ONNX/TensorRT/edge server deployment: 전처리와 detector를 합친 end-to-end FPS를 측정합니다.
  • Hanwha Vision camera, RTSP, and PTZ integration: RTSP 지연과 ONVIF PTZ 반응시간을 함께 기록합니다.
