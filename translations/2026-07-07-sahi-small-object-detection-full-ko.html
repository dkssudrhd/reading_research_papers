번역자 메모

번역자 메모: SAHI는 실무 적용 가치가 매우 높고 코드 라이선스도 permissive하지만, 논문 본문 전체 번역을 공개 저장소에 재배포할 권한은 별도 문제입니다. 따라서 공개 repo에는 경고 보고서와 원문 링크만 둡니다.

실무 읽기 메모

  • bird detection: 4K 광각 영상에서 원거리 새를 놓치지 않으려면 입력 축소와 tile 기반 처리의 차이를 실험합니다.
  • small/tiny object detection: tile size, overlap, merge NMS threshold가 핵심 튜닝 대상입니다.
  • PTZ camera tracking: SAHI는 PTZ가 움직이기 전 후보 탐지 recall을 높이는 데 유용합니다.
  • real-time RTSP/video analysis: 전 프레임 slicing은 비용이 크므로 selective slicing부터 검토합니다.
  • YOLO training/data design: tile label 변환과 원본 좌표 복원을 재현 가능한 스크립트로 고정합니다.
  • ONNX/TensorRT/edge server deployment: tile batch 수와 GPU memory peak를 detector export보다 먼저 계산합니다.
  • Hanwha Vision camera, RTSP, and PTZ integration: Hanwha Vision 광각 RTSP에서 후보를 찾고 PTZ는 안정적 track에만 반응하게 합니다.
