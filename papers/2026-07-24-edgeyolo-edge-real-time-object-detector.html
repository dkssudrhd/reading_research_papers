EdgeYOLO: 엣지 실시간 객체 탐지기

작은 새처럼 작고 드문 객체를 다루면서도 엣지 장치의 전체 지연을 줄이기 위한 anchor-free 탐지기 설계다.

원문(arXiv) · PDF · 공식 코드

전문 번역: arXiv 페이지에서 공개 번역·파생물 재배포를 허용하는 명시 라이선스를 확인하지 못해 공개 저장소 전문 번역은 보류했다.

한눈에 보는 요약

구조
anchor-free와 가벼운 decoupled head로 후처리 부담을 줄인다.
학습
유효 객체 상자를 더 보존하는 Mosaic+MixUp 조합과 hybrid random loss를 쓴다.
결과
COCO AP 50.6, VisDrone AP 26.4를 보고했고 Jetson AGX Xavier에서 30 FPS 이상을 목표로 한다.

논문 수치는 원문의 COCO/VisDrone 및 해당 장치 조건 결과다. RTSP 디코딩, 추적, PTZ 명령을 포함한 현장 FPS로 해석하면 안 된다.

왜 골랐나

금요일의 배포·경량화 주제에 정확히 맞고, 조류충돌방지에서 중요한 작은 객체 손실, 후처리 지연, GPU가 아닌 엣지 장치의 병목을 한 논문에서 연결한다. 공식 코드·가중치·하이퍼파라미터 링크가 있어 YOLO baseline과 바로 비교할 수 있다.

핵심 아이디어

자작 도식: 원문 그림을 복사하지 않았다.

Anchor-free: grid마다 여러 anchor를 검사하는 부담을 덜어 post-process 시간을 낮춘다. Decoupled head: 분류와 위치 회귀를 분리해 수렴과 회귀 품질을 개선한다. 유효 상자 보존 증강: 객체가 너무 작거나 잘린 합성 샘플을 억제한다.

기술 세부

후처리
논문은 전처리는 스트리밍 중 겹칠 수 있지만 후처리는 FPS에 포함해야 한다고 지적한다.
작은 객체
확대·splicing 기반 증강과 작은 객체에 더 주목하는 loss를 결합한다.
재매개변수화
학습 구조를 배포 구조로 합쳐 추론을 줄이는 설계를 사용한다.

L = L_cls + L_box + L_hybrid처럼 분류·상자·작은 객체에 민감한 손실의 균형을 검증해야 한다. 정확한 계수와 구현은 원문 및 공식 저장소 설정을 기준으로 삼는다.

조류충돌방지 적용

PTZ 추적
confidence만으로 움직이지 말고 box 크기, track age, 화면 가장자리 여부로 PTZ 명령을 gate한다.
YOLO 학습
동일 split에서 EdgeYOLO와 YOLO를 비교하고 16/32 px 이하 recall을 별도 집계한다.
실시간 서버
RTSP decode→inference→NMS→tracker→ONVIF/PTZ의 p50/p95를 분리 기록한다.

Hanwha Vision 카메라는 RTSP profile의 해상도·bitrate·timestamp를 고정하고, PTZ 이동 중 blur 구간은 탐지 threshold와 tracker update를 별도로 조정한다. ONVIF 또는 카메라 API의 pan/tilt 속도·명령 주기도 측정 대상이다.

즉시 실행할 실험 3개

  1. 공정 benchmark: 같은 조류 split·960 입력에서 YOLO와 EdgeYOLO의 AP, tiny-bird recall, false alarm/hour, end-to-end p95를 비교한다.
  2. label 밀도 증강: 일반 Mosaic와 유효 상자 보존 증강을 비교해 16 px 이하 recall·무효 label 비율을 기록한다.
  3. 폐루프 PTZ: 녹화 RTSP와 실제 카메라에서 30분씩 구동해 중심 오차, 재포착 시간, false command/hour를 측정한다.

용어

영문한국어 설명
Anchor-free미리 정한 여러 anchor shape 대신 위치 중심으로 box를 예측하는 탐지 방식.
Decoupled head분류와 box 회귀의 마지막 계산 경로를 분리한 head.
Post-processingscore filtering·NMS·좌표 복원처럼 모델 뒤에 수행되는 처리.
Re-parameterization학습 때의 여러 연산을 배포 때 하나로 합치는 최적화.

최종 체크리스트

  • □ 모델 FPS가 아니라 decode부터 PTZ까지의 p95 지연을 측정한다.
  • □ 작은 새의 크기 구간별 recall과 false alarm/hour를 분리한다.
  • □ augmentation 뒤 무효/잘린 label 비율을 표본 검수한다.
  • □ PTZ 이동 중 blur와 명령 지연을 detector 성능과 분리한다.
  • □ 공개 전문 번역 전에는 원문 라이선스를 다시 확인한다.
