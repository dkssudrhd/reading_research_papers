# Daily Paper Reading Notes

조류충돌방지 시스템, 컴퓨터 비전, 작은 객체 탐지, PTZ 카메라 추적, 실시간 영상 분석 관련 논문을 평일마다 1편씩 읽고 정리하는 저장소입니다.

## 주요 주제

- Bird Detection
- Small Object Detection
- Multi-Object Tracking
- PTZ Camera Tracking
- Wind Farm Bird Monitoring
- Real-time Computer Vision

## 파일 구성

- `index.html`: 매일 읽은 논문 목록과 핵심 메타데이터
- `papers/`: 날짜별 HTML 논문 요약 노트
- `papers.json`: 논문 목록을 재사용하기 위한 구조화 데이터

## 운영 방식

평일마다 업무와 연구에 직접 도움이 되는 논문 1편을 선정하고, 논문 내용을 한국어로 쉽게 요약합니다. 각 노트는 원문을 그대로 번역하지 않고 핵심 아이디어, 실무 적용 가능성, 바로 해볼 수 있는 실험을 중심으로 정리합니다.
